# YA Calculator

Production-ready calculator for fruit and vegetable wholesalers.

## Features

- **Screen 1 — Case Price:** Calculate unit price from case cost and quantity
- **Screen 2 — Unit Price:** Calculate sales total from unit price × quantity
- **Screen 3 — Running Total:** Add line items, undo, and clear with persistent storage

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- PWA support with service worker
- LocalStorage persistence

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Install as PWA

On iPhone Safari, tap Share → Add to Home Screen.
